# Orders Data Pipeline

A modular data engineering pipeline built with **Python, PostgreSQL, Docker, and Pandas**.  
This project demonstrates schema design, ETL ingestion of messy CSV/JSONL files, and analytics views for business insights.

---

## 🚀 Features
- **Schema Initialization**: Creates normalized tables for customers, orders, and order_items.
- **ETL Process**: Cleans and loads messy source files into PostgreSQL.
- **Analytics Views**: Provides reusable SQL views for top customers, monthly revenue, category breakdowns, and order status distribution.
- **Dockerized Database**: PostgreSQL runs in a container for easy setup and portability.
